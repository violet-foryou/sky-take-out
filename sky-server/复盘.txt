day 2 分页查询
1.query格式（紧跟在URL的 ? 后面，常用于get，用@RequestParam，只有一个对象接收时可以不用）和json格式（通常在HTTP请求的请求体 中，用@RequestBody，用于Post，PUT、PATCH）
2.log.info("进行分页查询，参数为{}",employeePageQueryDTO);,使用{}
3.converters.add(0,Converter);记得要写0，把新增转换器放在第一个
4.来自源根的路径：相对于classpath根目录，配置文件中应该使用的路径，比如配置文件中指明xml文件的路径，mapper-locations: classpath:mapper/*.xml
注意：

day 2 启用禁用员工账号
1.<if test="name != null and name != ''">XML 属性值本身通常用双引号包裹，如果在属性值内部再使用双引号，会导致 XML 解析错误，所以这里用单引号
2.name != null and name != '' 这个条件的意思是："只有当name参数被传递且传递的值不是空字符串时，才添加这个查询条件（null和''的区别）
null → "没有参数" 或 "参数未传递"
'' → "参数传递了，但值是空的"
